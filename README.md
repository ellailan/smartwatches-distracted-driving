# Smartwatches and Distracted Driving
This repo contains Exploratory Data Analysis of ["Smartwatches are more distracting than mobile phones while driving"](https://www.sciencedirect.com/science/article/abs/pii/S0001457520316663?via%3Dihub) by Brodeur et al. (2021)

> Completed as Case Study One for UBC's DSCI200 Course: Navigating Data: Acquisition, Exploration and Management
